# Trail Completionist

Simple Python GUI for tracking bike trail/route completion in an area

## Installation

### From source

```bash
pip install -e .
```

### Development Installation

For development, install with dev dependencies:

```bash
pip install -e ".[dev]"
```

## Development

### Running Tests

```bash
pytest
```

### Code Formatting

```bash
black src/ tests/
```

### Linting

```bash
flake8 src/ tests/
mypy src/
```
